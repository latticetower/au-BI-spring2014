Задача 17:
Дан текст T и N строк. Написать программу, которая для каждой строки находит все ее вхождения в текст, 
используя BWT. Ограничения: алфавит - {A,C,G,T},  |T| < 10^6, длины строк < 10^2, N < 10^4, время 2s.  
Входные данные: два fasta файла, в первом текст T, во втором строки, которые надо найти. В результате 
выдайте файл, в котором для каждой строки выписаны все позиции в тексте, с которых начинается искомая строка.
